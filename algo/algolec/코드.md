```python

```



`input.split(' ')` -> split(' ')에 ' '를 넣을 필요 없다. 기본값이 ' '(공백)임



파이썬에서 수가 커지면 오류가 날 수도 있다

즉, n! % mod 를 할 때,

1 * 2 * 3 * 4 * ... % mod 보다, 1 * 2 % mod * 3 % mod * ... 가 더 빠를 것이다.
