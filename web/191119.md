1. Basic box model

   ```css
   .header{
       background-color: yellow;
       width: 1000px;
       height : 100px;
       border :10px solid #f00;
       padding: 0 10px;
       margin: 0 0 0 50px;	/*위아래좌우*/
           
   }
   ```

   - border : width에 더해짐
   - padding도 마찬가지
   - margin도 더해진다
   - ex) 1000px + 10px + 10px+ 50px -> 실제 크기
