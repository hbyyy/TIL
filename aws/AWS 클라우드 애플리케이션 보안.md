# AWS 클라우드 애플리케이션 보안

## AWS 책임 공유 모델

- AWS는 고객의 데이터를 가지고 있다.

### 클라우드의 보안

- AWS는 AWS 리전, 가용 영역, 엣지 로케이션등의 글로벌 인프라를 보호한다.
- AWS의 최우선 과제



## 서비스

### 비관리형 서비스

- EC2, EBs

### 관리형 서비스

- RDS, S3, DynamoDB

### 운영

- 게스트 os 패치
- 데이터베이스 패치
- 방화벽 구성

## 인증 및 권한 관리

### IAM

- AWS 리소스에 대한 액세스를 안전하게 제어
  - IAM 사용자
    - AWS와 상호 자용하는 사람 또는 애플리케이션
  - 그룹
    - 동일한 권한을 가진 IAM사용자 모음
  - 역할
    - 엔티티가 맡을 수 있는 임시 권한

### 인증 : 사용자 확인

- CLI, SDK, management console을 통해 IAM 인증 가능
